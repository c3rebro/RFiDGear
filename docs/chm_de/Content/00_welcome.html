Willkommen zur integrierten Hilfe der RFiDGear Kodiersoftware.

Systemvoraussetzungen:

  1. PC-Hardware: Einen PC, der die Mindestvorraussetzungen für das Betriebssystem Windows 7 (32bit/64bit) erfüllt.
  2. Betriebssystem: Windows 7 (32bit/64bit) oder höher
  3. Peripherie: Mindestens ein PC/SC kompatibles Programmiergerät.
  4. Unterstützte Herstelle sind z.B.: HID, Sciel, ACR und weitere.

RFiDGear ist eine Kodierungssoftware für RFID Medien (im Folgenden als "Chip" bezeichnet). Kern der Software sind "Aufgaben" welche folgende Eigenschaften besiten können:

  1. Einen Aufgabenindex (Zahlenwert) der die Reihenfolge bestimmt, in der die Aufgaben ausgeführt werden.
  2. Eine frei wählbare Bezeichnung.
  3. Ein Ergebnis in Form eines "Fehlercodes", welcher bei der Ausführung der Aufgabe entsteht, und intern in der Aufgabe für spätere Verwendung gespeichert wird.
  4. Eine Ausführungsbedingung. Diese ist Optional und besteht aus dem Index und dem erwarteten Fehlercode einer anderen -bereits ausgeführten- Aufgabe.

Die erstellten Aufgaben können durch einzelanwahl, als sammelauftrag oder vollautomatisch bei erkennen eines neuen Chips abgearbeitet werden. Die Aufgaben werden sequentiell abgearbeitet. Hierbei kann es sich um eine der folgenden Aufgabentypen handeln:

  1. Chiptypunabhängige Aufgaben:
    • Ausfüllen einer PDF-Forms Vorlage mit selbst definierten Texten oder Variablen.
    • Logikbausteine für "Wenn-Dann" Prüfungen.
  2. Chipbezogene Aufgaben:
    • Generische Aufgaben
      • UID Prüfen.
      • Chiptyp prüfen.
    • Mifare Aufgaben
      • Mifare Classic
        • Daten lesen
        • Daten schreiben
        • Sektor-"unbenutzt" Prüfung
      • Mifare Desfire
        • Prüfung: App existenz
        • PICC: Masterkey ändern
        • PICC: Formatieren
        • App: Erstellen
        • App: Schlüssel ändern
        • App: Löschen
        • App: Authentifizieren
        • Datei: Erstellen
        • Datei: Schreiben
        • Datei: Lesen
        • Datei: Löschen
  3. asd
  4. asd
